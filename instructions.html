Afin d'apprendre les technologies web la meilleur maniere est la methode des trois P : Pratique Pratique Partique. Pour cela de nombreuses questions sont mise a disposition dans la section des tests rapides. Ces tests ne sont pas note, vous pouvez donc en faire autant que vous voulez sans couler votre cours. Lorsque vous serez fin pret, vous pourrez effectuer l'examen de la technologie que vous voulez. Afin d'effectuer un test rapide ou un examen vous devez:

  • Rendez vous a la page du tableau de bord
  • Selectionner la technologie
  • Selectionner le nombre de questions si vous voulez effectuer un examen
  • Cliquez sur le type de test que vous voulez effectuer
